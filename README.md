# 🛍 Tilda → VK Webhook Bot

Сервис для автоматической отправки уведомлений о новых заказах с сайта на Tilda в личные сообщения VK.

Когда покупатель оформляет заказ на сайте — все нужные сотрудники мгновенно получают красиво оформленное сообщение в VK.

---

## ✨ Что умеет

- Принимает данные из форм Tilda через Webhook
- Форматирует заказ в читаемое сообщение с эмодзи
- Отправляет уведомление сразу нескольким получателям в VK
- Поддерживает разные способы оплаты (наличные, Tinkoff, Сбербанк и др.)
- Поддерживает разные типы доставки (самовывоз, Яндекс Доставка и др.)

## 🚀 Запуск локально

### 1. Клонируй репозиторий

```bash
git clone https://github.com/ВАШ_НИК/tilda-vk-bot.git
cd tilda-vk-bot
```

### 2. Создай файл `.env`

```env
VK_TOKEN=твой_токен_сообщества_VK
VK_RECIPIENTS=123456789,987654321
```

Где:
- `VK_TOKEN` — токен сообщества VK (Управление → Работа с API → Создать ключ)
- `VK_RECIPIENTS` — список VK ID получателей через запятую

### 3. Установи зависимости и запусти

```bash
pip install -r requirements.txt
uvicorn main:app --reload
```

Сервер запустится на `http://localhost:8000`

### 4. Проверь через curl

```bash
curl -X POST http://localhost:8000/webhook \
  -H "Content-Type: application/x-www-form-urlencoded; charset=UTF-8" \
  --data-urlencode "Name=Иван Иванов" \
  --data-urlencode "Phone=+7 (999) 123-45-67" \
  --data-urlencode "Email=test@test.com" \
  --data-urlencode "paymentsystem=tinkoff" \
  --data-urlencode 'payment={"orderid":"123","products":["Товар (00001), pc=500"],"amount":"500","subtotal":"500","delivery":"Самовывоз","delivery_price":0,"delivery_fio":"Иван Иванов","delivery_city":"Москва","delivery_address":"","delivery_comment":""}'
```

## 🐳 Запуск через Docker

```bash
docker build -t tilda-vk-bot .
docker run -p 8000:8000 --env-file .env tilda-vk-bot
```

## ☁️ Деплой на Railway

1. Форкни репозиторий
2. Зайди на [railway.app](https://railway.app) и создай новый проект из GitHub репозитория
3. В разделе **Variables** добавь переменные окружения:
   - `VK_TOKEN`
   - `VK_RECIPIENTS`
   - `PORT` = `8000`
4. Railway автоматически соберёт Docker-образ и задеплоит сервис
5. В разделе **Settings → Networking** нажми **Generate Domain**
6. Вставь полученный URL в настройки Webhook на Tilda

---

## 🔗 Настройка Webhook в Tilda

1. Открой редактор сайта → нужный блок с формой
2. Настройки формы → **«После отправки»** → **Webhook**
3. Вставь URL: `https://ВАШ_ДОМЕН.up.railway.app/webhook`
4. Метод: **POST**
5. Сохрани и опубликуй сайт
